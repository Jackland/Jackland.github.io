第1节：swoole基础学习

swoole 学习记录

如何学习

    1，多看文档
    2，多看案例代码
    3，多书写代码

swoole简介

    1，PHP异步多线程通信引擎（原生php不支持多线程）

了解swoole 的

    1，tcp  ==》server （服务器）
    swoole :worker_num //最大进程数 

    开启客户端 telnet 127.0.0.1 9501

    2，tcp ==》client(客户端)

php源码安装（安装准备 php7 swoole2.1 redis Ubuntu )

    php.net 获取php源码  安装 gcc库 
    1，解压 （php.ini  源码安装默认在安装目录lib 底下） php -i |grep php.ini
    2，configure (编译脚本 安装源代码库) ./configure --prefix=安装路径
    3，make
    4，make install

swoole 源码安装

    1，码云下载 编译安装 使用phpize 生成 configure文件

websocket（服务端）

    http协议通信只能有客户端发起  服务器返回结果（http 轮询可以做到 服务器主动推送信息）
    建议tcp协议上 持久化的网络通信协议 
    集成 swoole_http_server

swoole进程讲解 master 进程（最近跑数据 要用到*）

进程就是已经运行的实例（process）
1，创建子进程 子进程之间的通信 需要管道（进程和进程之间的桥梁） 子进程的pid 和 主进程的pid
2，查看 进程  netstat -anp |grep 8811 
3， 查看进程树 pstree -p  进程号
4，查看进程 ps aft | grep 服务名
5 将内容输出到管道中去 ，使用$swoole->write()写入管道   $swoole->read 方法获取管道内容

swoole 内存管理模块之一 (swoole_table)
